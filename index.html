    The Modular Ocean Model, MOM6

    Project maintained by NOAA-GFDL Hosted on GitHub Pages — Theme by mattgraham

    Welcome to GitHub Pages for MOM6.

    MOM6 is currently under active development. If you are looking for the most mature ocean model from GFDL, then you might be looking for MOM5, which is supported at https://mom-ocean.github.io, and which you can find at http://www.gfdl.noaa.gov/ocean-model.

    MOM6 documentation can be found in several key locations:

    • Installation and getting started - goto the MOM6-examples wiki.
    • Explore experiments and ipython notebooks - goto the MOM6-examples repository.
    • Read MOM6 developer guides - goto the MOM6 wiki.
    • View the MOM6 code documentation - view the doxygen output for MOM6 APIs.

    Source code:

    • All repositories live under the NOAA-GFDL GitHub organization.
    • See MOM6 source code - goto the MOM6 repository.
    • See FMS source code - goto the FMS repository.
